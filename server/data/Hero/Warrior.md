# 战士 (Warrior)

战士是游戏中的近战基础职业，拥有较高的初始生命值和物理攻击力。

## 基础属性

我们在这里定义战士的基础属性变量：
{{var: BaseHP=500}}
{{var: BaseAtk=45}}
{{var: BaseDef=30}}

- **生命值**: {{ref: BaseHP}}
- **物理攻击**: {{ref: BaseAtk}}
- **物理防御**: {{ref: BaseDef}}

## 成长属性 (10级)

当战士达到10级时，生命值将得到大幅提升。
计算公式：基础生命值 + 10级成长附加值 (300) = 最终生命值
{{calc: BaseHP + 300}}
