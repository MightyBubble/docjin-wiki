# Docjin Wiki 简介

欢迎使用这款专为游戏设计与原型展示打造的本地 Markdown Wiki！

## 核心特性体验指南

我们现在使用最简洁的 `{{ }}` 语法来定义和引用变量！Vditor 会自动将它们渲染为对应的 UI 面板，点击面板即可无缝查看/修改源码。

1. 打开 `Hero/Warrior.md` 
   - 体验：利用 `{{var: 变量名=值}}` 定义局部变量。
   - 体验：利用 `{{ref: 变量名}}` 和 `{{calc: 公式}}` 引用和计算变量。
2. 打开 `Demo/CombatSystem.md` (重点推荐！)
   - 体验：跨文件引用 `Warrior.md` 和 `Mage.md` 中的变量属性。
   - 体验：利用跨文件引用的变量进行混合公式计算（伤害计算）。
   - 体验：利用 `{{embed: 文件路径}}` 实现跨文档级别的全文或指定标题嵌入。

> **提示**：所见即所得编辑器下，光标移出后会自动渲染成最终结果的精美胶囊小卡片。鼠标点击它们可以再次还原成文本进行编辑！保存时它们会完美还原为纯文本的 Markdown！
